# Calculando Financiamento

Simulador de financiamento em Python + Flask.  
Permite calcular parcelas com base em salário, valor do imóvel e entrada, respeitando o limite de 30% do salário.

## Tecnologias
- Python 3.10+
- Flask
- HTML + CSS

## Como rodar
1. Clone o repositório:
   ```bash
   git clone https://github.com/seuusuario/Calculando-Financiamento.git

## Instalação do Flask
